## Featured Projects

### [Platform CI/CD System](https://github.com/Prayasvadlakonda/platform-cicd-system)

A production-style CI/CD pipeline built with GitHub Actions and Docker.

- Multi-stage pipeline: test → build → deploy (dev → prod)
- Artifact promotion using immutable commit-SHA image tags — build once, deploy everywhere
- Manual approval gates before production releases
- Rollback simulation wired in for failure scenarios

**Stack:** Python · FastAPI · Docker · GitHub Actions

---

### [AWS Infrastructure Platform](https://github.com/Prayasvadlakonda/infra-platform-aws)

Terraform modules for deploying containerized applications on AWS.

- ECS Fargate behind an Application Load Balancer inside a private VPC
- Separate dev and prod environments, each with isolated networking
- Remote state via S3 + DynamoDB locking ready to enable
- Per-environment CloudWatch log groups for observability

**Stack:** Terraform · AWS ECS · ALB · VPC · CloudWatch · IAM
